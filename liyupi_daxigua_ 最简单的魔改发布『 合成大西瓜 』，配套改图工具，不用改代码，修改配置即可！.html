ローカルで開始

2つのローカル起動方法が提供されています。serveとDockerです。Xiaobaiは最初の方法をお勧めします。

初心者アプリケーション

  1. サーブツールをインストールします。

    npm i-gサーブ

  2. daxiguaディレクトリに入り、serveを実行します。

    サーブ

  3. ブラウザを開き、localhost：5000にアクセスしてください。

Dockerがインストールされています

おかげでの貢献Buchenglei

  1. イメージを構築する

    docker build -tdaxigua-server 。

  2. コンテナを起動します

    docker run -d --name play-daxigua -p5000：5000 daxigua-server

クイックマジックチェンジ

以下の手順に従って変更し、追加を続けます

  1. スコアを変更します：extraSettings.jsファイルを変更します

  2. 画像を変更する：res / raw-assetsディレクトリの下の指定されたディレクトリにある画像を置き換えます。このディレクトリには、同じファイル名、サフィックス、サイズが必要です。正常な上書きが有効になり、マテリアルファイルを置き換えることができます。

  3. 無敵モード：extraSettings.jsファイルを変更します

  4. 最初のフルーツを指定します：extraSettings.jsファイルを変更します

  5. 次回表示されるフルーツを指定します。extraSettings.jsファイルを変更します

  6. 大きな果物から小さな果物の合成：extraSettings.jsファイルを変更します

  7. フルーツの応答性を高める：extraSettings.jsファイル、参照原則を変更する

  8. 果物の落下速度が遅くなりました：extraSettings.jsファイル、参照原則を変更してください

  9. 替换音乐：，覆盖 res/raw-assets 目录下相同的音乐，可替换素材文档

  10. 替换背景：和改图片原理相同，可替换素材文档

  11. 去广告：将广告图片替换为同背景色底图

  12. 替换广告链接：改 extraSettings.js 文件

  13. 改网站标题：改 extraSettings.js 文件

  14. 开启选分弹窗：改 extraSettings.js 文件

  15. 点击右上方图标更换水果: 改 extraSettings.js 文件

上线发布

提供了多种上线发布方式，腾讯云一键部署、Vercel、腾讯云静态网站托管、GitHub Pages 等，小白建议使用 Vercel。

腾讯云一键部署

适用于会 fork 仓库、提交代码的同学。

点击下方按钮，无需任何操作，即可部署我的仓库到腾讯云 ⬇️

如果要替换为一键部署自己的仓库，要先 fork 我的仓库，本地修改后再提交到自己 fork 的仓库，最后把一键部署按钮仓库地址改为自己的仓库即可！

Vercel

适用于零基础的小白。操作简单、域名简短，但国外的服务器，比较慢。

Vercel 是免费网站托管平台，可以帮我们部署网站，并生成可访问的简短网址，还能够和自己购买的域名进行绑定。

先在命令行通过 npm 命令安装 Vercel：

npm install -g vercel

安装完成后，进入 index.html 所在目录（我的是 daxigua），使用 vercel 命令发布网站：

cd daxigua
vercel --prod

然后会让你输入一些选项，比如项目名称、是否和已有项目关联、是否保存当前配置等。如果要创建多个项目，千万不要和已有项目关联！

发布成功，会得到一个网址，打开就可以看到游戏啦，还可以把网址分享给别人！

问题及解决

  1. 无法安装 serve 或者 Vercel？

    答：如果报找不到 npm，请先安装 npm。

    如果安装中卡住，试着按下键盘（可能假死），还不行的话先用 npm 安装 cnpm（国内镜像，比较快）：

    npm install cnpm -g --registry=https://registry.npm.taobao.org

    再用 cnpm 安装： cnpm i -g serve 或 cnpm i -g vercel

  2. Vercel 网址被微信拦截怎么办？

    答：可以把网址复制到浏览器打开，也可以申请一个域名，在 Vercel 和服务提供商配置域名解析。 Vercel 基本是海外的服务器，无需备案。

  3. 怎么在电脑上浏览网页游戏？

    答：在浏览器中，按 F12 打开开发者工具，点击像手机一样的图标即可。

  4. 为什么 serve 后，打开网页一片空白？

    答：大概率是你在错误的目录下执行了 serve，请务必在 index.html 所在的文件夹下执行 serve。

  5. 执行 vercel 命令显示 signUp？

    答：要先去 Vercel 官网 注册。

  6. vercel 邮箱验证失败？

    答：先确认邮箱是否正确，如果验证失败，大概率是网络原因，请尝试 4G 等网络。或者在其他浏览器中，打开邮箱，点击验证按钮。

  7. 怎么使用 vercel 同时上线多个版本？

    答：在输入 vercel 后，选择不和已有项目关联（link），并且使用一个新的项目名（project name）。

  8. ファイルを変更して新しいバージョンを作りたいのですが、vercel prodと入力すると、既存のプロジェクトとリンクするかどうかを選択せず​​に上書きされます。どうすればよいですか？

    A：ベルセルを実行した後.vercel、隠しディレクトリを保存する前にローカルに公開された情報が生成されます。ディレクトリを削除してください。

  9. URLをエクスポートした後、画像を変更しましたが、元の画像はまだゲーム内にありますか？

    回答：Webサイトはリモートファイルを読み取るため、ローカルファイルのみを変更することは無意味です。vercelまたはTencentCloudコマンドを再度実行し、最新のファイルをアップロードします。

  10. Macはこのチュートリアルを使用できますか？

    回答：もちろんです！すべてのコマンドはウィンドウとまったく同じです！cmdコマンドラインツールを変更して、ターミナルターミナルを使用するだけです（コマンド+スペースを押して、ターミナルを検索します）

  11. Webサイトが空白の画面になるのはなぜですか？

    回答：間違いを犯し、一部のファイルが失われる可能性が高いです。。コードをもう一度ダウンロードして変更してみてください。公開する前に、ローカルで実行できることを確認してください。
